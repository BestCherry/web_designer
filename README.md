### 介绍: 
* 项目使用了`Vue`进行开发
* 网页设计器图形化工具,目前内置`ElementUi`组件库
* 支持上传自定义组件,一键生成网页代码,支持进行二次开发

### 项目运行
```
npm install
```

### 编译开发
```
npm run serve
```

### 编译打包
```
npm run build
```
